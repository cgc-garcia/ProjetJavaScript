
# Mini-Projet Cinéma

Application web utilisant l'API OMDb.

## Installation
```bash
npm install
npm start
```

## Pages
- index.html : films tendances
- search.html : recherche en temps réel
- movie.html : détails d’un film

## API
Créer un fichier `src/js/config.js` avec :
```js
export const API_KEY = "VOTRE_API_KEY";
```
